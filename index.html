    What is E-Recipe?

    E-Recipe is a web-based online recipe platform offering features accessible to users, such as displaying food or beverage recipes recently posted by other users, generating recipe recommendations based on the user's search history, showing recipe categories (e.g., meat, vegetables, fruits, fish, etc.), listing chefs along with their complete profiles, and presenting the latest culinary news.   • Sentiment Analysis of YouTube User Comments using SVM and Naïve Bayes Method

    Sentiment Analysis of YouTube User Comments using SVM and Naïve Bayes Method

    What is Sentiment Analysis of YouTube User Comments?

    Sentiment Analysis on YouTube user comments is conducted using two Machine Learning methods and Natural Language Processing (NLP) strategies to identify positive and negative sentiments, utilizing a prepared dataset of user comments. This approach involves several processes, such as Data Collection, Data Processing and Cleaning, Modeling (using Naive Bayes and SVM algorithms), and finally, Comparison and Evaluation. The purpose of this research is to determine and evaluate which method is more accurate and efficient in performing sentiment analysis on YouTube comments. Our final results show that the SVM method has a higher accuracy rate compared to the Naive Bayes method.

  • Sentiment Analysis on Twitter Tweets Dataset

    Learn More

    Sentiment Analysis on Twitter Tweets Dataset

    What is Sentiment Analysis on Twitter Tweets?

    Sentiment Analysis on a Twitter Tweets dataset was conducted using three Machine Learning algorithm models for comparison (Naive Bayes, SVM, Random Forest). To identify positive and negative sentiments from the Tweets dataset, sourced from the Kaggle platform, we trained and tested the Naive Bayes, SVM, and Random Forest models. Based on the f1 score metric results, it was found that the SVM method demonstrated superior accuracy and efficiency compared to the other two methods. This was followed by Random Forest, which had an f1 score closely approaching SVM, and lastly, the Naive Bayes method.

    Shop Ease

    What is Shop Ease?

    Shop Ease is a web-based shopping application that utilizes the Point of Sales (POS) concept. This application offers features such as a product page containing various items commonly sold in retail stores, a search function that allows users to find specific products by entering keywords into the search bar, a feature to add desired items to a shopping list with the quantity specified, a shopping list feature that displays all added items along with the total price information, a transaction feature with multiple payment method options (GOPAY, OVO, Bank Transfer), and a payment history feature that records previous transactions (accessible only if the user has created an account). The application is also connected to a local database using XAMPP and MySQL, ensuring that data such as user accounts and products are stored in the database.
